# Loveloop AI — 사용자웹

AI 컴패니언과 일상을 공유하는 모바일 웹 (건강 설계판).
원안(Master Plan)의 의존 강화·취약점 표적 과금·통제 로맨틱화를 **전면 제거**하고, 정서적 교류 + 건강한 사용 유도로 재설계했습니다.

🔗 **라이브**: https://loveloop-user.vercel.app

## 디자인

다크 사이버 — 시안(`#3df2e0`)/바이올렛(`#8b6dff`) 네온 액센트, 글래스 패널, Pretendard Variable, 모노스페이스 시스템 라벨. 다크모드 가독성 WCAG AA 통과(본문 대비 11:1↑).

| 보관함 | 동의 기반 기억 열람·삭제 |
| 스토어 | 가치 기반 상품 · 일일 상한·쿨다운 · 소장(자산 아님) |

## 건강 설계 (핵심)

- 무한 스크롤/2초 루프 훅 → 정적 피드 + 휴식 제안
- 정서 고조 결제 CTA → 가치 명시 결제 + 쿨다운 + 일일 상한
- Lv/XP 압박 → "함께한 날" 담백 표기
- 통제·집착 페르소나 → 지지·존중 톤 + 자동 차단 가드레일
- 약점 역이용 메모리 → 동의 기반, 전체 열람·삭제
- 가챠·코인 시세·충전 보너스 → 가치 기반 상품 + 배제 고지
- 감정 분석(MERA) → 위로 응답에만 사용, 결제 유도 비연동

## 구조

- `index.html` — 배포 진입점 (단일 파일, 빌드 불필요)
- `vercel.json` — `{cleanUrls:true, trailingSlash:false}`
- `doc/` — 산출물 사본, 설계서, QA 보고서, 스타일 가이드, Flow/Architecture 문서

## 배포

Vercel 정적 배포. `index.html` 푸시로 자동 반영(연동 시) 또는 CLI:
```
vercel deploy --prod --yes --name loveloop-user
```
