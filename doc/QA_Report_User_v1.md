## 4. 배치/구조 분석 (화슈 관점)

현재 6 view(홈·대화·페르소나·웰빙·보관함·스토어)는 안전 설계엔 충실하나,
마스터플랜이 강조한 **멀티모달 경험의 깊이**가 평면적이다.

개선 배치 제안:
1. **톱바**: 페르소나 아바타 + 상태 + (신규) 언어/지역 칩 + 알림
2. **홈**: 안부 히어로 → **예약된 안부 카드(능동적 기억 표면화)** → 주간 통계 → 피드 → 휴식
3. **대화**: 텍스트 버블 + (신규) **영상통화 진입 + 통화 화면**(MERA 감정 반영, 위로 중심)
4. **페르소나/웰빙/보관함/스토어**: 현행 유지 + 사실감 라벨 보완
